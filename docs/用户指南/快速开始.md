# 快速开始

欢迎使用图像处理工具箱！这是一个专为浏览器设计的图像处理工具，提供图像压缩和 SVG 转图片功能。

## 前置要求

### 系统要求

- **浏览器**：Chrome 88+、Edge 88+、Opera 74+ 或其他基于 Chromium 的浏览器
- **操作系统**：Windows、macOS、Linux 均可使用
- **网络**：无需网络连接，完全本地处理

### 安装插件

#### 在 Chrome 中安装

1. 下载或克隆此项目到本地
2. 打开 Chrome 浏览器
3. 在地址栏输入 `chrome://extensions/` 并回车
4. 开启右上角的「开发者模式」开关
5. 点击「加载已解压的扩展程序」
6. 选择 `图像处理游览器插件` 项目根目录
7. 插件将成功安装并出现在浏览器工具栏中

#### 在 Edge 中安装

1. 打开 Edge 浏览器
2. 在地址栏输入 `edge://extensions/` 并回车
3. 开启「开发人员模式」
4. 点击「加载解压缩的扩展」
5. 选择项目目录并安装

#### 在 Opera 中安装

1. 打开 Opera 浏览器
2. 在地址栏输入 `opera://extensions/` 并回车
3. 开启「开发者模式」
4. 点击「加载已解压的扩展」
5. 选择项目目录并安装

## 基础使用

### 打开插件

1. 在浏览器工具栏中找到插件图标
2. 点击图标打开主页面
3. 选择要使用的功能：
   - **图像压缩**：压缩和优化图片
   - **SVG 转图片**：将 SVG 转换为 PNG 或 JPG

### 图像压缩

#### 上传图片

1. 选择「图像压缩」功能
2. 拖拽图片到上传区域
3. 或点击上传区域选择文件
4. 支持格式：PNG、JPG/JPEG、WebP、GIF、BMP、PPTX

#### 调整参数

1. **压缩质量**：拖动滑块调整（1-100 级）
   - 推荐值：80 级（平衡质量和大小）
2. **尺寸调整**：选择预设或输入自定义值
   - 自动：保持原始尺寸
   - 75%、50%、25%：按比例缩小
   - 1920px、1280px：限制长边像素数
3. **输出格式**：选择目标格式
   - PNG：无损压缩，适合透明背景
   - JPG：有损压缩，适合照片
   - WebP：现代格式，体积小
   - GIF：适合简单动画
   - BMP：无损格式，文件较大

#### 执行压缩

1. 点击「开始压缩」按钮
2. 等待处理完成
3. 查看压缩结果：
   - 原始文件大小
   - 压缩后大小
   - 压缩率
   - 图片预览

#### 下载结果

1. **单个下载**：点击每个结果的「下载」按钮
2. **批量下载**：
   - 勾选需要下载的文件
   - 点击「全选」可选中所有结果
   - 点击「批量下载」按钮
3. 查看下载进度和完成状态

### SVG 转图片

#### 添加 SVG

1. 选择「SVG 转图片」功能
2. 通过两种方式添加 SVG：
   - **方式一：文件上传**
     - 拖拽 SVG 文件到上传区域
     - 或点击上传区域选择文件
   - **方式二：代码粘贴**
     - 直接在文本框中粘贴 SVG 代码
     - 点击「添加到任务列表」按钮
3. 支持批量添加多个 SVG 文件

#### 调整参数

1. **输出格式**：选择 PNG 或 JPG
2. **分辨率倍数**：
   - 1x（标准）：保持原始尺寸
   - 2x（2K）：放大 2 倍
   - 3x（4K）：放大 3 倍
   - 4x（8K）：放大 4 倍
3. **图片质量**（仅 JPG 格式）：
   - 1-10 级质量调节
   - 数值越高质量越好

#### 执行转换

1. 点击「批量转换」按钮
2. 查看转换进度：
   - 当前处理的文件名
   - 已完成/总任务数
   - 当前任务进度
   - 总体进度条

#### 下载结果

1. 所有转换完成后，点击「批量下载」按钮
2. 所有转换的图片将自动下载
3. 查看下载管理器确认完成

## 高级功能

### PPTX 文件处理

#### 自动提取图片

1. 上传 PPTX 演示文稿文件
2. 插件自动提取所有嵌入的图片
3. 自动排除系统图片和过小图片
4. 自动去重（通过文件哈希识别重复）

#### 批量压缩

1. 提取的图片自动进入压缩队列
2. 可以统一应用压缩设置
3. 支持批量下载所有图片

### 多语言支持

#### 切换语言

1. 点击右上角语言切换按钮
2. 选择：
   - **中文**：简体中文界面
   - **English**：英文界面
3. 界面即时切换，无需刷新页面

### 主题切换

#### 切换主题

1. 点击右上角主题切换按钮
2. 选择：
   - **浅色主题**：明亮界面
   - **深色主题**：黑暗界面，护眼
   - **跟随系统**：自动跟随操作系统主题
3. 主题偏好自动保存

## 常见问题

### 问题：图片无法上传

**解决方案：**
- 检查文件格式是否支持（PNG、JPG、WebP、GIF、BMP、PPTX）
- 检查文件大小是否超限（最大 100MB）
- 尝试拖拽而非点击选择

### 问题：压缩后图片质量太差

**解决方案：**
- 提高压缩质量（使用 80-100 级）
- 避免过度压缩
- 选择合适的输出格式（PNG 适合透明图片）

### 问题：SVG 转换失败

**解决方案：**
- 检查 SVG 代码是否完整
- 尝试简化复杂的 SVG 代码
- 检查是否有不支持的 SVG 特性

### 问题：下载失败

**解决方案：**
- 检查浏览器下载权限
- 尝试在下载管理器中查看
- 清除浏览器缓存后重试
